# JS30

is a series of projects built with vanilla JS based on [JavaScript 30 by WesBos](https://github.com/wesbos/JavaScript30).

1. **Drum kit** play with PC keyboard. See [demo](https://ming-yong.github.io/JS30/01-drum-kit/) and [notes](./notes/01-drum-kit.md).
2. **Wall clock** with digits under hands and a circle showing day/night on the center. See [demo](https://ming-yong.github.io/JS30/02-clock/) and [notes](./notes/02-clock.md).
